
The BSIM-CMG 107.0.0 package contains the following files:

/README.txt
This document

/Guileline_document.pdf
Guideline document for changes done to BSIM-CMG106.1.0.
  
/BSIM-CMG107.0.0 Technical Note.pdf
The BSIM-CMG Technical Note (or user's manual).
 
/How_to_switch_off_models_in_BSIMCMG.txt
Instructions on turning off (deactivating) various modules in BSIM-CMG.

/code/bsimcmg.va
Verilog-A file pointing to bsimcmg_main.va for testing convenience

/code/common_defs.include
Macros and constant definitions

/code/bsimcmg_main.va
Main Verilog-A file 

/code/bsimcmg_body.include
Main Verilog-A include file 

/code/bsimcmg_nqsmod3.va
Main Verilog-A for NQSMOD=3

/code/bsimcmg_compute_nqsmod3_segment.include
Include file for NQSMOD=3

/code/bsimcmg_load_nqsmod3_segments.include
Include file for NQSMOD=3

/code/bsimcmg_nqsmod3_iv_cv.include
Include file for NQSMOD=3

/code/bsimcmg_quasi_static_cv.include
Include file for quasi-static IV, CV

/code/bsimcmg_cfringe.include
Include file for geometric fringe cap model

/benchmark_test/*.sp
Sample netlists for HSPICE

/benchmark_test/*.lis
Sample output files

/benchmark_test/modelcard.nmos
Sample model card for nmos

/benchmark_test/modelcard.nmos.1
Another sample model card for nmos

/benchmark_test/modelcard.pmos
Sample model card for pmos

/benchmark_test/modelcard.pmos.1
Another sample model card for pmos

/benchmark_test/clean_all.sh
Unix Shell script to clean un-needed simulation output files

/benchmark_test/run_all.sh
Unix Shell script to run all the benchmark tests
